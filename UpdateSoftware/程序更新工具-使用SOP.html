程序更新工具 V2.0 · 使用 SOPEXE版本管理

适用范围：工控机工位 EXE 程序版本扫描 / 匹配 / 批量更新 · 内部工具操作手册
本工具用于统一管理工控机各工位的 EXE 程序版本：可扫描当前版本、对比新旧版本、一键批量更新。日常推荐使用「智能操作」页完成更新，仅需 3 步。

操作流程总览

① 登录→ ② 参数设定（首次）→ ③ 设置新版本路径→ ④ 一键扫描→ ⑤ 一键更新→ ⑥ 监听模式（可选）

一、界面总览

主窗口为三栏式布局：顶部工具栏 + 左侧目录树 + 右侧四个功能 Tab 页。
图1 · 主界面三栏式布局总览（默认显示「智能操作」Tab）

二、核心操作步骤（日常使用）

1
首次启动 · 选择关闭行为
首次运行弹出设置对话框，选择窗口关闭时的行为。建议选「最小化到系统托盘」（蓝色按钮），后续可在「参数设定」中修改。
图2 · 首次运行设置对话框
2
身份验证 · 登录
主窗口默认锁定（半透明遮罩 + 橙色锁图标）。点击顶部「身份验证」按钮（橙红色），输入用户名密码登录，遮罩消失即可操作。
图3 · 锁定状态与身份验证登录
3
参数设定（仅首次配置）
切换到「参数设定」Tab，配置工位文件夹识别规则与 EXE 名称匹配规则。配置完成后点击顶部「保存参数」，再点「读取配置」生效。
图4 · 参数设定页（首次配置工位规则与路径）
此步骤仅需在首次部署或工位规则变更时操作，日常更新无需重复配置。
4
智能操作 · 设置路径并一键扫描
切换到「智能操作」Tab（★推荐主流程）。设置新版本统一存放路径（选外层总目录，自动搜索子文件夹 EXE），确认扫描模式为「远程」，点击「一键扫描所有工位」。
图5 · 智能操作页：设置路径 → 一键扫描
5
智能操作 · 查看对比并一键更新
扫描完成后查看结果表格：绿色=最新、红色=可更新、灰色=未检测到新版本。确认后点击「一键更新所有过期工位」（橙色，扫描后启用）完成批量更新。
图6 · 扫描结果对比与一键更新
6
监听模式 · 自动更新（可选）
开启「启用监听模式」后，工具自动监视新版本目录；检测到新 EXE 文件时，自动扫描所有工位并执行更新，无需人工干预。
图7 · 监听模式（开启后自动更新）
监听模式适合长期运行场景；如需手动控制更新时机，保持关闭即可。

三、进阶操作（按需使用）

以下两个 Tab 为手动操作方式，适合单工位调试或特殊场景。日常批量更新请优先使用「智能操作」。
程序扫描（Tab 1）
用于手动扫描指定工位文件夹的程序版本。在「新设备程序目录配置」表格中添加行、填写 IP/共享名/账户/密码等信息后，点击「创建程序目录」生成目录，再点「扫描选中文件夹程序版本」。勾选「深度搜索」可扫描子目录；勾选「过滤旧版本结果」可只显示最新版本（绿色）。
程序匹配（Tab 2）
用于为每条扫描记录手动指定待推广的程序路径。在「程序扫描」页勾选记录后点「获取所选信息」，数据会带入本页表格；为每行填写「待推广程序路径」（可浏览或拖拽），确认版本对比后点「更新(复制-替换匹配-重命名)」执行更新。

四、状态颜色速查

颜色含义出现位置
● 绿色最新版本（A=B 或已是最新）扫描结果 / 匹配表格
● 红色可更新（A<B，存在新版本）扫描结果 / 匹配表格
● 灰色未检测到新版本智能操作扫描结果

五、常见问题与注意事项

页面打不开 / 全部灰色？ → 未登录。点击顶部「身份验证」按钮登录，遮罩消失后即可操作。
扫描后「一键更新」按钮仍是灰色？ → 说明没有检测到可更新工位（全部为最新或灰色），或未勾选待更新行。确认结果中有红色「可更新」记录。
扫描不到工位程序？ → 检查「参数设定」中的工位文件夹名称规则与 EXE 名称是否与实际一致；确认扫描模式选择正确（远程/本机）。
新版本路径怎么填？ → 选择外层总目录即可，系统自动搜索其下所有子文件夹内的 EXE，无需逐个指定。
设备号前缀有什么用？ → 勾选「启用设备号」并填写后，创建目录/扫描时会以「设备号#」作为工位名称文件夹前缀，便于多设备区分。
参数改了不生效？ → 修改参数设定后，必须依次点击「保存参数」→「读取配置」才会生效。
{版本号} 占位符？ → 参数设定表格中的 {版本号} 会在运行时自动替换为实际版本号，无需手动填写。
关闭窗口后程序还在运行吗？ → 若首次选择「最小化到系统托盘」，关闭窗口后程序驻留托盘；可在「参数设定」底部修改此行为。
